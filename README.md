# harnesskit-resources

Community-maintained resource files consumed by [HarnessKit](https://github.com/anthropics/harnesskit).

## Structure

### `tips/`

Curated tips displayed inside the HarnessKit CLI. See [`tips/TIPS_GUIDELINES.md`](tips/TIPS_GUIDELINES.md) for contribution rules.

### `cli-registry/`

Machine-readable registry of third-party CLIs available in the HarnessKit marketplace. See [`cli-registry/REGISTRY_GUIDELINES.md`](cli-registry/REGISTRY_GUIDELINES.md) for the entry schema and how to add new CLIs.
